# 🛍️ Haircare Brand Availability Analysis – Bangalore Quick Commerce

## 📌 Project Overview

This project analyzes the availability of the top 10 haircare brands across leading quick commerce platforms in Bangalore.

Platforms analyzed:

* Blinkit
* Zepto
* Swiggy Instamart

Pin Codes covered:

* 560038 – Indiranagar
* 560034 – Koramangala
* 560066 – Whitefield
* 560102 – HSR Layout
* 560041 – Jayanagar

The objective was to structure SKU-level data and derive analytical insights on brand availability, pricing, discount trends, and stock consistency.

---

## 📊 Dataset Structure

The Excel file contains three sheets:

### 1️⃣ Raw Data

Each row represents one product (SKU) for a specific:

* Brand
* Platform
* Pin code

Captured fields:

* Product Name
* MRP (₹)
* Selling Price (₹)
* Discount (%)
* Stock Status
* Estimated Delivery Time (minutes)
* Date & Time of Collection

---

### 2️⃣ Brand Summary

Brand-level analysis including:

* Total Entries per Brand
* Platforms Available
* Pin Codes Available
* Average Discount (%)
* Out-of-Stock Count

All metrics are dynamically calculated using Excel formulas.

---

### 3️⃣ Platform Comparison

Platform-level performance analysis:

* Total Brands Available
* Total SKUs
* Average Discount (%)
* Out-of-Stock Rate (%)

---

## 🧠 Key Insights

* Pantene offers the highest average discount (~40%), indicating aggressive promotional pricing.
* Indulekha shows minimal discounting, suggesting premium positioning.
* Swiggy Instamart provides the highest average discount among platforms.
* All selected brands were available across platforms during the data collection window.
